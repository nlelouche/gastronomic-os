I. Resumen Ejecutivo: Arquitectura de Datos GOS-100
El recetario GOS-100 no es una simple colección de platos, sino un Grafo de Decisiones Clínicas diseñado para resolver el conflicto entre eficiencia logística y seguridad médica en el hogar22.
+1


1. Seguridad Clínica de Precisión: Cada receta integra protocolos binarios de exclusión para 18 condiciones, incluyendo la técnica de Lixiviación de 4 horas para pacientes renales (eliminación de potasio) y el Protocolo SIGHI para la gestión de histamina3333.
+1

2. Lógica de Ramificación (Branching): El sistema identifica automáticamente los "Universal Bases" (pasos aptos para todos) y los "Branch Points" (puntos de bifurcación), permitiendo cocinar una base común y separar porciones personalizadas solo cuando es estrictamente necesario4.

3. Sustitución Funcional: Se han mapeado más de 400 variantes lógicas utilizando ingredientes como el "Huevo de Lino", el Aquafaba y las Leches de Granos para mantener la integridad culinaria (textura/sabor) sin violar restricciones inmunológicas5555.
+1

4. Optimización Metabólica: Todas las recetas incluyen parámetros de control glucémico para Diabetes (uso de fibra estructural) y carga de carbohidratos netos para perfiles Keto6666.
